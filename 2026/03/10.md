## わかったこと
- stateはユーザーの動きに合わせて変わる値のこと
- stateはオブジェクト"{プロパティ名:値}"で記述する
- stateの初期値を変更する際は"this.setState(オブジェクト)"
- propsはApp.jsからコンポーネントに渡すデータ
- HTMLとJSXの記法の違い
- Express: Node.jsでWebアプリの開発をするためのフレームワーク。パッケージの一つ。
- パッケージ: 便利な機能を簡単に使えるようにまとめたもの
- npm: node package manager, パッケージを共有・取得する機能がある
- ejs: embedded java script
