## わかったこと

-- CLI --
- s3のバケット作るときのコマンドをAWSのサイトから調べようとしたが、create-s3やcreate-buで調べても出てこず、Webブラウザからaws cli s3で調べたら、awsのコマンドのページで出てきた。それでもmbがバケットを作成するコマンドであると最初はわからなかった。
 
-- SDK --
- AWS SDK for Python (boto3)　←Python用のSDKがboto3
- プレデンシャルは権限のこと
- EC2への操作には権限が必要となるがアクセスキー発行するやり方かIAMロール設定してやるやり方があり、推奨は後者。アクセスキー発行は権限が広すぎてセキュリティーリスクが非常に高いため。

-- IAM --
- AWS Identity and Access Management - IDとAWSサービス/リソースへのアクセスを安全に管理するサービス
- セキュリティーの中核を担うサービスでAWSアカウント自体のセキュリティを設定する
- 構成要素としてはIAMポリシー、IAMユーザー、IAMグループ、IAMロール
- AWSにおいてセキュリティーは最優先事項と言われている
- IAMポリシーの設定方法
