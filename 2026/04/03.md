## わかったこと
- Amazon Elastic Compute Cloud = Amazon EC2は使いたいときに使いたいだけ使える仮想サーバー、OSより上のレイヤーを自由に構築可能
- EC2は秒単位の従量課金
- Elastic IPアドレスを設定しインスタンスに関連付けることで公開グローバルIPが固定される。ただし、現在は停止中でも設定されたままだと課金されてしまうようになった。
- AMI（Amazon Machine Image）はEC2インスタンスを作るもとになる。
- EBSはElastic Block Storeの略でディスクボリュームのこと
